# Future_DS_01
E-Commerce Sales Performance Dashboard (Power BI)
Project Overview

This project focuses on building an interactive E-Commerce Sales Performance Dashboard using Power BI. The dashboard is designed to help analyze sales, profit, and order trends across different regions, categories, and time periods, enabling data-driven business decisions.

Objectives

Analyze overall sales, profit, and order volume

Identify monthly sales trends

Understand category-wise and region-wise performance

Highlight top-selling products

Provide interactive filtering for better insights

Tools Used

Power BI Desktop

Microsoft Excel

DAX (Data Analysis Expressions)

Key Features

KPI cards for Total Sales, Total Orders, and Total Profit

Monthly sales trend using a line chart

Sales distribution by product, category, and region

Interactive slicers for year, region, and category

Clean and consistent dashboard layout

Approach

Imported and cleaned sales data in Power BI

Created calculated measures using DAX

Built interactive visuals to answer key business questions

Applied design best practices for readability and clarity

Insights

Sales follow clear monthly trends with seasonal peaks

Technology category generates the highest revenue

Certain products significantly outperform others

West and East regions contribute the most to total sales

Learning Outcomes

Data cleaning and preparation

KPI creation using DAX

Dashboard design and layout optimization

Business insight generation using Power BI
